# Task 3 - Exact Register Map

| Offset | Register   | Access | Description |
|--------|------------|--------|-------------|
| **0x00** | **GPIODATA** | **R/W** | Output data register |
| **0x04** | **GPIODIR**  | **R/W** | Direction (1=output) |
| **0x08** | **GPIOREAD** | **R**   | Pin state readback |
